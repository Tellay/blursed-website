`CSS`, ou (Cascading Style Sheets) é uma linguagem de `estilos` cujo a sua utilização é adicionar estilos a uma página <Link href="/blog/html">web</Link>.

## Utilização

Pode ser utilizada `internamente` no arquivo de `HTML` ou `externamente` utilizando o elemento `<link>`

### Exemplo de utilização internamente
```html showLineNumbers {6-17}
<!DOCTYPE html>
<html>
  <head>
    <title>Título</title>
    
    <style>
        * {
            padding: 0;
            margin: 0;
            box-sizing: border-box;
        }

        body {
            background-color: black;
            color: white;
        }
    </style>
  </head>
  <body>
    <h1>O meu primeiro cabeçalho.</h1>
    <p>O meu primeiro parágrafo.</p>
  </body>
</html>
```

### Exemplo de utilização externamente

```html {6}
<!DOCTYPE html>
<html>
  <head>
    <title>Título</title>
    
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>O meu primeiro cabeçalho.</h1>
    <p>O meu primeiro parágrafo.</p>
  </body>
</html>
```

```css title="styles.css" showLineNumbers
* {
    padding: 0;
    margin: 0;
    box-sizing: border-box;
}

body {
    background-color: black;
    color: white;
}
```
