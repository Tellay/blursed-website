`HTML`, ou HyperText Markup Language, é a linguagem de `marcação` cuja função é `estruturar` e `organizar` todas as páginas da `web`.

## What is HTML for?

Como já dito antes é usada para criar documentos na web desde simples páginas `estáticas` a sites complexos e `dinâmicos`.
Permite ao desenvolvedores especificar o layout, texto, imagens, links e outros elementos que fazem parte de uma página web.

## HTML and CSS

`HTML` tem por defeito estilos muitos simples e com aparência antiga. Por isso usamos `CSS` (Cascading Style Sheets) para estilizar os elementos de uma página.

## HTML and Interactivity
Por defeito `HTML` também não tem interatividade para isso usamos uma linguagem de programação chamada `JavaScript`. Todos os sites no planeta utilizam desta linguagem para que posso haver interatividade e funcionalidades nas páginas web.
