`HTML`, ou HyperText Markup Language, é a linguagem de `marcação` cuja função é `estruturar` e `organizar` todas as páginas da `web`.
A sua `primeira` versão foi lançada em 1991 pelo britânico `Tim Berners-Lee`.

## Para que é usada?

Como já dito antes é usada para criar documentos na web, desde simples páginas `estáticas` a sites complexos e `dinâmicos`.
Permite ao desenvolvedores especificar o layout, texto, imagens, links e outros elementos que fazem parte de uma página web.

## Estilos

`HTML` tem por defeito estilos muitos simples e com aparência antiga. Por isso usamos <Link href="/blog/css">CSS</Link> (Cascading Style Sheets) para estilizar os elementos de uma página.

## Interatividade

Por defeito `HTML` também não tem interatividade para isso usamos uma linguagem de programação chamada `JavaScript`. Todos os sites no planeta utilizam desta linguagem para que haja interatividade e funcionalidades nas páginas web.

### Estrutura básica

```html showLineNumbers
<!doctype html>
<html>
  <head>
    <title>Título</title>
  </head>
  <body>
    <h1>O meu primeiro cabeçalho.</h1>
    <p>O meu primeiro parágrafo.</p>
  </body>
</html>
```
