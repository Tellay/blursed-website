C# é uma linguagem de alto nível, multiparadigma e fortemente tipada. Foi criada em 2000 por Anders Hejlsberg na Microsoft.

## Para que é usado?

Pode ser utilizado para vários funções, tais como:

- Desktop
- Mobile
- Jogos

### Exemplo de código

```cs showLineNumbers
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World!");
        }
    }
}
```
