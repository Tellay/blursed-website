Base de dados, de forma simples são `coleções de dados` armazenados em um `sistema informático.` Um exemplo muito simples de umas base de dados analógica pode ser a sua lista de compras escrita em um papel.

## Tipos de bases de dados

Existem vários tipos de bases da dados, sendo que estes param mim são os mais conhecidos:

- Relacionais (SQL)
- Não relacionais (NoSQL)

## MySQL

MySQL é uma base de dados `open-source` e `gratuita`. Utiliza da linguagem SQL (Structured Query Language).

## MongoDB

MongoDB é uma base da dados `NoSQL` também open-source. Esta é orientada a documentos muito semelhantes a JSON.

## Microsoft Access

Microsoft Access é um software de `gerenciamento` de `base de dados` produzido pela Microsoft. Faz parte do pacote `Microsoft Office` e oferece uma `interface gráfica` para `criar` e `gerenciar` bases de dados. Teve a sua primeira versão lançada em novembro de 1992.

- Acessável atráves de: [Microsoft](https://www.microsoft.com/pt-pt/microsoft-365/access)

## Tabelas

Uma tabela é organizada em colunas e linhas. Cada `coluna` é designada por `campo` e cada `linha` por `registo`.
É importante que a tabela **não** tenha entradas repetidas.

Cada registo contém sempre um ou mais campos. Estes têm de ser designados como um determinado tipo de dados, seja `texto`, `data`, `hora`, `número` ou outro tipo.

## Formulários

Os formulários têm `botões de comando` e `controlos` para desempenhar tarefas. Estes também permitem controlar a maneira como os utilizadores `interagem` com a base de dados.

## Relatórios

De forma muito simples, relatórios são usados para `formatar`, `resumir` e `apresentar` dados. Pode ser personalizado à gosto da pessoa.

## Consultas

Consulas têm como a sua função `consultar` dados específicos. Permite adicionar critérios para `filtrar` os dados de acordo com aquilo que desejamos.
