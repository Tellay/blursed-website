C++ é uma linguagem de programação de alto nível criada em 1979 por Bjarne Stroustrup na Bell Labs como uma extensão do C.

## Para que é usado?

Pode ser utilizado para vários funções, tais como:

* Desktop
* Jogos
* Sistemas Operativos
* Compiladores e Interpretadores de outras linguagens
